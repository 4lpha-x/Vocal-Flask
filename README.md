# Vocal-Flask,
A Open Source Tool To Clone Any Website On LocalHost For Testing.

This Tool Works On Termux Properly

Any One Can Configure It For Any Linux Distribution,

Requirements : Python3, Git

Commands To Clone This Tool On Termux :
```
apt update
apt upgrade -y 
pkg install git
pkg install python
git clone https://github.com/HACKE-RC/vocal-flask
cd vocal-flask
```
